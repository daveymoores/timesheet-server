timesheet-gen

Installation

> brew install timesheet-gen

Initialise for current directory

> timesheet-gen init [run mode: -cb / -d]

Initialise for specified repository

> timesheet-gen init /path/to/repo

Edit timesheet for today

> timesheet-gen edit -h 5

Remove entry for a specific day

> timesheet-gen remove -d 4

Generate timesheet

> timesheet-gen make

Generate timesheet for specific month

> timesheet-gen make -m 12
